# Barbie-Themed UX/UI Portfolio

A minimalist, interactive, and playful single-page portfolio website inspired by the Barbie aesthetic.
Built with modern web technologies for a seamless and responsive user experience.

## âœ¨ Features

- **Barbie Aesthetic**: Vibrant pinks, glassmorphism, and playful micro-interactions.
- **Interactive Elements**:
  - Custom cursor that reacts to hoverable elements.
  - Floating 3D shapes and parallax effects.
  - Draggable sticker board playground.
  - Color palette generator.
- **Smooth Animations**: Powered by **Framer Motion** for buttery smooth transitions.
- **Sections**:
  - **Hero**: Dynamic gradient background and typewriter effect.
  - **About**: Animated profile with skills badges and fun facts.
  - **Projects**: Masonry grid with category filters and detailed modal.
  - **Playground**: Interactive mini-games.
  - **Contact**: Animated form with confetti success effect.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.

## ðŸš€ Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/barbie-portfolio.git
    cd barbie-portfolio
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
